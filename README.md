# AI ChatBot Java

## Description
A simple AI-inspired chatbot developed using Java. The chatbot responds to user inputs using predefined rules and provides interactive conversations.

## Features
- User interaction
- Rule-based responses
- Simple AI logic
- Exit functionality

## Technologies Used
- Java
- Scanner Class

## How to Run
1. Open the project in Eclipse.
2. Run ChatBot.java.
3. Start chatting with the bot.
